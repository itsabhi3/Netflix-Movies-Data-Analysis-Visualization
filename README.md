# 🎬 Netflix Movie Data Analysis

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a Netflix Movies dataset to discover trends, patterns, and insights from movie-related data.

The analysis includes data cleaning, genre analysis, popularity analysis, movie trends over the years, and visualization of key findings using Python libraries.

---

## 🛠️ Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Project Workflow

### 1. Data Cleaning
- Removed unnecessary columns
- Handled missing values
- Checked duplicate records
- Converted data into appropriate formats

### 2. Exploratory Data Analysis (EDA)
- Genre analysis
- Popularity analysis
- Year-wise movie analysis
- Visualization of trends and distributions

### 3. Data Visualization
Created multiple charts and plots using:
- Bar Charts
- Count Plots
- Histograms
- Pie Charts
- Heatmaps

---

## 🔍 Key Findings

### Q1: What is the most frequent genre in the dataset?
- Drama is the most frequent genre in the dataset.
- It represents more than 14% of all movies.

### Q2: Which genre has the highest votes?
- Drama received the highest number of votes.
- More than 18.5% of popular movies belong to the Drama genre.

### Q3: Which movie has the highest popularity?
- Spider-Man: No Way Home has the highest popularity score.
- Genres:
  - Action
  - Adventure
  - Science Fiction

### Q4: Which movie has the lowest popularity?
- The United States, Thread has the lowest popularity score.
- Genres:
  - Music
  - Drama
  - War
  - Sci-Fi
  - History

### Q5: Which year produced the most movies?
- Year 2020 recorded the highest number of movie releases in the dataset.
